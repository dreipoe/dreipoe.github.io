Минимальные системные требования
================================

* Операционная система: Microsoft Windows 7 SP1, ReactOS 0.4.7
* Процессор Intel Pentium 4
* Оперативная память:
	* 192 МБ для ReactOS
	* 1 ГБ для Windows x86
	* 2 ГБ для Windows x64
* Microsoft SQL Server 2012
* Клавиатура, мышь

Возможна работа ПО на платформах, не удовлетворяющих данным требованиям, однако стабильная работа не гарантируется.
